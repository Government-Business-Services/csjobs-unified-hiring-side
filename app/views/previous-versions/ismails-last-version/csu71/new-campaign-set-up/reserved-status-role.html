Reserved status

In addition to the rules that all candidates must satisfy to be eligible to join the Civil Service, there are some roles that can only be filled by UK nationals.

These roles are reserved because of the sensitive nature of the work carried out. For example, all individuals working in any of the security and intelligence services will be employed into a post with a reserved status.
