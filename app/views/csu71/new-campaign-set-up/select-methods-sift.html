How will you assess at sift?

If you receive lots of applications and need to set up an initial sift

You can set up an initial sift on the next page, choosing a lead method to assess applications.

When you have sifted using this initial method, the remaining candidates then progress to full sift.

If you do this, the assessment methods for both sifts must be shown in your job advert.
